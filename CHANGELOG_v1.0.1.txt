CHANGELOG - Version 1.0.1
=========================

## New Features
- Operator dapat mengelola siswa (tambah, edit, hapus, assign ke kelas)
- Import siswa dari Excel/CSV untuk operator
- Template CSV untuk import siswa
- Auto-detect APP_URL untuk deployment yang lebih fleksibel

## Improvements
- Perbaikan .htaccess untuk kompatibilitas Apache 2.2 dan 2.4
- Router dengan path detection yang lebih dinamis
- Error handling yang lebih baik untuk database connection
- Perbaikan routing untuk deployment di berbagai environment

## Bug Fixes
- Fix Internal Server Error saat deployment di aaPanel
- Fix APP_URL hardcoded yang menyebabkan masalah di production
- Fix .htaccess RewriteBase yang tidak fleksibel

## Technical Changes
- Update config/config.php dengan auto-detect APP_URL
- Update router.php dengan path detection yang lebih baik
- Update .htaccess dengan syntax yang kompatibel
- Perbaikan error handling di database.php

## Files Added
- operator/manage_siswa.php - Halaman manage siswa untuk operator
- operator/manage_siswa_template.php - Template CSV untuk import
- CHECKLIST_INSTALASI.txt - Checklist instalasi
- admin/naik_kelas.php - Fitur naik kelas
- api/version_management.php - API untuk version management
- guru/ujian/delete.php - Fitur hapus ujian
- migration_add_is_operator.php - Migration untuk is_operator

## Files Modified
- config/config.php - Update versi ke 1.0.1, auto-detect APP_URL
- router.php - Perbaikan path detection
- .htaccess - Perbaikan kompatibilitas
- index.php - Perbaikan redirect
- operator/index.php - Tambah menu manage siswa
- includes/header.php - Update menu operator
- Dan file lainnya

## Migration Notes
- Tidak ada migration database yang diperlukan
- Pastikan PHP version >= 7.4
- Pastikan mod_rewrite aktif di Apache
- Untuk import Excel, install PhpSpreadsheet (opsional, CSV tetap bekerja)

## Breaking Changes
- Tidak ada breaking changes

## Deployment Notes
- File .htaccess sudah diupdate untuk kompatibilitas
- APP_URL sekarang auto-detect, tidak perlu diubah manual
- Pastikan file permissions benar (644 untuk file, 755 untuk folder)
